## Projetos Acadêmicos

### **• Sistema de Concessionária (Java)**
Aplicação para simular a gestão de uma concessionária, com cadastro de veículos, clientes e serviços, além do controle de vendas. Focado em orientação a objetos e boas práticas de modelagem.

### **• BasketBallStore (Excel + VBA)**
Sistema de loja criado no Excel com automações em VBA. Possui cadastro de produtos, controle de estoque e registro de vendas, com foco em automação e organização de dados.

### **• Agenda (React + Node.js)**
Projeto full stack para gerenciamento de contatos, com frontend em React (formulários dinâmicos, exibição e edição de dados) e backend em Node.js com Axios. Desenvolvido com foco em integração entre front e back-end.
