Aplicação de Fila de Atendimento

Desenvolvi um sistema em Java para gerenciar a fila de atendimento em um laboratório de coleta de sangue, aplicando conceitos básicos de Estrutura de Dados. O sistema foi implementado sem o uso de bibliotecas específicas como ArrayList ou Queue, utilizando estruturas personalizadas como listas e filas simples. O projeto tem como objetivo otimizar o processo de atendimento e garantir a organização na fila de pacientes, aplicando técnicas de programação e lógica de estrutura de dados.
